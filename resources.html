Key Resources for Implementing FAIR²

Explore these guides, tools, and platforms to help you implement the FAIR² framework, ensuring your data is accessible, AI-ready, and ethically managed.

  • FAIR² Implementation Guide:
    A comprehensive guide on how to apply the FAIR² principles to your research data. Download the guide
  • Croissant Metadata Format:
    Learn how to use the Croissant metadata format to make your data AI-ready and standardized for machine learning workflows. Croissant Metadata Details
  • SHACL Validation Tool:
    Validate your metadata structure and ensure it meets the FAIR² standards with SHACL validators. SHACL Validation Tool
  • MLCommons Tools:
    Access Python-based and web-based tools from MLCommons for managing and validating metadata. MLCommons Tools
  • QUDT Ontology:
    Standardize your dataset units using the Quantities, Units, Dimensions, and Types (QUDT) ontology. QUDT Ontology
  • PROV Ontology for Provenance Tracking:
    Track the origins and changes in your data with the PROV ontology for detailed provenance information. PROV Ontology
  • SENSCIENCE AI Data Steward:
    Learn more about the AI-powered tool for automating metadata creation and responsible data management. AI Data Steward Tool
  • Zenodo Repository for FAIR² Datasets:
    Explore datasets adhering to the FAIR² framework in the open-access Zenodo repository. Zenodo Repository
